## P1219	\[USACO1.5\] 八皇后 Checker Challenge

八皇后问题, 输入棋盘大小, 输出解的个数

### 思路

dfs 维护四个数组.
- 每行的棋子放在第几列 $row$
- 每列的棋子放在第几行 $col$
- 两个对角线, 对角线可以用 $x+y=a$ 和 $x-y=a$ 表示

由于 python 运行速度慢, 大数据点可以通过**打表**过.

## P1443	马的遍历

有一个 $n \times m$ 的棋盘，在某个点 $(x, y)$ 上有一个马，要求你计算出马到达棋盘上任意一个点最少要走几步。

### 思路

bfs 模板题, python 可以使用 `queue.Queue()` 预置的队列.
